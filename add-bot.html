Add a New Telegram Bot

Telegram Bot Token Kaise Le? (How to get Telegram Bot Token)

  1. Telegram app open karein aur search karein @BotFather.
  2. @BotFather ko open karke /start likhein.
  3. /newbot command bhejein aur apne bot ka naam aur username set karein.
    (Username hamesha bot pe end hona chahiye, jaise: mytestbot)
  4. Bot create hote hi aapko bot token mil jayega. Yeh kuch aisa dikhega:
    123456789:ABCdefGhIJKlmNoPQRstuVWXyz
  5. Yeh token yahan form me Bot Token field me paste karein.
  6. Bot ko apne Telegram group ya channel me add karne ke liye, usko admin rights bhi de sakte hain (agar zarurat ho).

English Guide

  1. Open Telegram app and search for @BotFather.
  2. Open @BotFather and type /start.
  3. Send /newbot command, then set your bot's name and username.
    (Username must end with bot, e.g., mytestbot)
  4. After creation, you will get a bot token like:
    123456789:ABCdefGhIJKlmNoPQRstuVWXyz
  5. Paste this token in the Bot Token field above.
  6. To add your bot to a group or channel, give it admin rights if needed.
